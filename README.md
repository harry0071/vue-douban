# 移动端豆瓣电影

> 一个基于vue.js + muse-ui的移动端豆瓣电影app，具有查看正在热映电影、电影排行、北美榜单、影评详情、搜索电影等功能。

## 技术栈
vue + vue-router + vue-jsonp + muse-ui + es6 

## 用法
运行前请先到[官网](https://nodejs.org/zh-cn/download/)安装node.js      
然后将项目clone到本地。           
执行下面的命令

``` bash
# 首先安装依赖
npm i

# 然后运行项目
npm start

# 打包
npm run build
```
